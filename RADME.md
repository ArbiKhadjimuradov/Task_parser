# Codeforces Task Parser

Этот проект парсит задачи с Codeforces и предоставляет Telegram-бота для поиска задач.

## Установка

1. Клонируйте репозиторий:
 - git clone <###>
2. Запустите проект с Docker:
- docker-compose up --build
## Использование
 - Парсинг задач: автоматически каждый час.

 - Telegram-бот: /search <сложность> <тема>.

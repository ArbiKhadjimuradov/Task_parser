# Codeforces Task Parser
# Бот для поиска задач с Codeforces по уровню сложности.
Этот проект парсит задачи с Codeforces и предоставляет Telegram-бота для поиска задач.

## Установка
- Клонируйте репозиторий:
- git clone <https://github.com/ArbiKhadjimuradov/Task_parser.git>


### 1. Установка зависимостей
Установите Docker и Docker Compose.

### 2. Создание Telegram-бота
1. Перейди в Telegram и найди BotFather (`@BotFather`).
2. Создай нового бота с помощью команды `/newbot`.
3. Скопируй токен бота.

### 3. Настройка окружения
Создай файл `.env` в корне проекта и добавь туда токен бота:
- TELEGRAM_TOKEN=ваш_токен_бота #`без 'ковычек'`!

### 4. Запуск приложения
Собери и запусти контейнеры.
Командами в терминале:

- docker-compose build
- docker-compose up

### 5. Использование бота
- Перейди в Telegram и найди бота которого создали.
- Используй команду /start, чтобы начать.
- Введи сложность задачи (например, 800) или сложность и лимит (например, 800 5).
- Примеры команд:

#### /start — начать работу с ботом.

#### 800 — получить 10 задач с рейтингом 800.

#### 800 5 — получить 5 задач с рейтингом 800.

#### 1200 — получить 10 задач с рейтингом 1200.
